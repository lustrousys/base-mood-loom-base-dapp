# Base Mood Loom

## App Name

Base Mood Loom

## Tagline

Weave today's mood onchain

## Description

Weave one daily mood onto Base with a color strand, a short reason, and a visual personal archive.

## Category

Mood / self-tracking / visual utility

## Short Pitch

Base Mood Loom turns a daily feeling into a visual strand on Base, giving people a more human and color-first way to keep a lightweight emotional archive.

## Production Environment Variables

- `NEXT_PUBLIC_MOOD_LOOM_CONTRACT_ADDRESS`: deployed mood loom contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
